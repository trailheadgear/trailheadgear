## 🚀 部署（3 步上线，免费）

### 方法 1：GitHub Pages（推荐，完全免费）

```bash
cd camping-site
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/你的用户名/你的仓库名.git
git push -u origin main
```
然后去 GitHub 仓库 → Settings → Pages → Source: main → Save。

### 方法 2：Netlify/Vercel

直接把 `camping-site/` 文件夹拖到 netlify.com 的部署页面。3 秒上线。
